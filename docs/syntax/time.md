# Time Expressions (Časové údaje)

Expressing time requires mastering several prepositions and cases.

## 1. Days (Accusative)
Use **v/ve** + Accusative.
*   *V pondělí* (On Monday)
*   *V úterý* (On Tuesday)
*   *Ve středu* (On Wednesday)
*   *Ve čtvrtek* (On Thursday)
*   *V pátek* (On Friday)
*   *V sobotu* (On Saturday)
*   *V neděli* (On Sunday)

## 2. Months (Genitive/Locative)
*   **In January**: *V lednu* (Locative: v + -u)
*   **In May**: *V květnu*
*   **In September**: *V září* (Locative = Nominative)

## 3. Clock Time (V kolik hodin?)
Use **v/ve** + Number.
*   *V jednu hodinu.* (At 1:00)
*   *Ve dvě hodiny.* (At 2:00)
*   *V pět hodin.* (At 5:00)

## 4. Duration (Jak dlouho?)
Use pure Accusative (no preposition).
*   *Čekám **jednu hodinu**.* (I have been waiting for one hour.)
*   *Byl tam **celý den**.* (He was there all day.)

## 5. Future/Past Offset (Za / Před)
*   **Za** + Accusative = "In X time" (Future)
    *   *Přijdu **za hodinu**.* (I will come in an hour.)
*   **Před** + Instrumental = "X time ago" (Past)
    *   *Byl jsem tam **před hodinou**.* (I was there an hour ago.)
