# Conditional Clauses (Podmínkové věty)

Conditional clauses usually start with **kdyby** (if) or **jestli** (if/whether).

## 1. Real Conditions (Jestli)
Used when the condition is possible or likely.
*   **Structure**: *Jestli* + Present/Future/Past Indicative.
*   *Jestli máš čas, přijď.* (If you have time, come.)
*   *Jestli bude pršet, zůstaneme doma.* (If it rains, we will stay home.)

## 2. Unreal Conditions (Kdyby)
Used for hypothetical situations (Present Conditional).
*   **Structure**: *Kdyby* + Past Tense (l-participle) + Conditional (bych/bys/by...).
*   *Wait, "kdyby" actually combines "když" + "by"!*

### Conjugation of Kdyby

| Person | Form |
| :--- | :--- |
| Já | **kdybych** |
| Ty | **kdybys** |
| On | **kdyby** |
| My | **kdybychom** |
| Vy | **kdybyste** |
| Oni | **kdyby** |

### Examples
*   *Kdybych měl peníze, koupil bych si dům.* (If I had money, I would buy a house.)
*   *Kdyby to věděl, neudělal by to.* (If he knew it, he wouldn't do it.)

*Common Mistake: Do NOT say "Kdybych by měl". The "by" is already inside "kdybych"!*
