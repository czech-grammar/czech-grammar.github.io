# Questions (Otázky)

## 1. Yes/No Questions
These are formed simply by **intonation** (raising your voice at the end) or by inverting the subject and verb (optional).

*   *Máš čas?* (Do you have time?)
*   *Je to pravda?* (Is it true?)

You can also use the particle **jestli** (if/whether) in indirect questions.
*   *Nevím, jestli přijde.* (I don't know if he will come.)

## 2. Wh-Questions
These start with an interrogative pronoun or adverb.

| Czech | English | Example |
| :--- | :--- | :--- |
| **Kdo?** | Who? | *Kdo je to?* |
| **Co?** | What? | *Co děláš?* |
| **Kde?** | Where? | *Kde bydlíš?* |
| **Kam?** | Where to? | *Kam jdeš?* |
| **Kdy?** | When? | *Kdy přijedeš?* |
| **Jak?** | How? | *Jak se máš?* |
| **Proč?** | Why? | *Proč pláčeš?* |
| **Který?** | Which? | *Který chceš?* |
| **Čí?** | Whose? | *Čí je to auto?* |
