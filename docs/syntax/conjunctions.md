# Conjunctions (Spojky)

Conjunctions link words or sentences.

## Coordinating Conjunctions
Link two equal elements.

*   **a** (and): *Petr a Pavel.*
*   **i** (and also): *Petr i Pavel.*
*   **ale** (but): *Je malý, ale silný.* (He is small, but strong.)
*   **nebo** (or): *Káva nebo čaj?* (Coffee or tea?)
*   **proto** (therefore): *Prší, proto nejdu ven.* (It's raining, therefore I'm not going out.)

## Subordinating Conjunctions
Link a main clause and a dependent clause.

*   **že** (that): *Myslím, že je to pravda.* (I think that it is true.)
*   **protože** (because): *Nejdu tam, protože nemám čas.* (I'm not going there because I don't have time.)
*   **když** (when/if): *Když prší, spím.* (When it rains, I sleep.)
*   **aby** (so that/in order to): *Jdu tam, aby viděl.* (I go there so that he sees.)

## Comma Rules (Briefly)
*   Before **a, i, nebo**: Usually **NO** comma.
*   Before **ale, že, protože, když, aby, který**: Always **YES** comma.
