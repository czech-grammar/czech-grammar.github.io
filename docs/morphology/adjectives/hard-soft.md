# Hard and Soft Adjectives

## 1. Hard Adjectives (Tvrdá)
The vast majority of Czech adjectives belong to this group. The dictionary form ends in **-ý**.

**Model**: *Mladý* (Young)

### Singular Declension

| Case | Masculine Animate | Masculine Inanimate | Feminine | Neuter |
| :--- | :--- | :--- | :--- | :--- |
| **Nom** | mlad**ý** | mlad**ý** | mlad**á** | mlad**é** |
| **Gen** | mlad**ého** | mlad**ého** | mlad**é** | mlad**ého** |
| **Dat** | mlad**ému** | mlad**ému** | mlad**é** | mlad**ému** |
| **Acc** | mlad**ého** | mlad**ý** | mlad**ou** | mlad**é** |
| **Voc** | mlad**ý**! | mlad**ý**! | mlad**á**! | mlad**é**! |
| **Loc** | mlad**ém** | mlad**ém** | mlad**é** | mlad**ém** |
| **Instr** | mlad**ým** | mlad**ým** | mlad**ou** | mlad**ým** |

### Plural Declension

| Case | Masculine Animate | Masculine Inanimate | Feminine | Neuter |
| :--- | :--- | :--- | :--- | :--- |
| **Nom** | mlad**í** | mlad**é** | mlad**é** | mlad**á** |
| **Gen** | mlad**ých** | mlad**ých** | mlad**ých** | mlad**ých** |
| **Dat** | mlad**ým** | mlad**ým** | mlad**ým** | mlad**ým** |
| **Acc** | mlad**é** | mlad**é** | mlad**é** | mlad**á** |
| **Voc** | mlad**í**! | mlad**é**! | mlad**é**! | mlad**á**! |
| **Loc** | mlad**ých** | mlad**ých** | mlad**ých** | mlad**ých** |
| **Instr** | mlad**ými** | mlad**ými** | mlad**ými** | mlad**ými** |

*Note: In Nom. Pl. Masc. Anim., hard consonants soften (d->ď, h->z, ch->š, k->c, r->ř). E.g., Český -> Čeští.*

## 2. Soft Adjectives (Měkká)
These adjectives end in **-í** in the dictionary form. They are "soft" because the ending **-í** stays the same for **all genders** in the singular Nominative.

**Model**: *Jarní* (Spring)

### Singular Declension

| Case | Masculine Animate | Masculine Inanimate | Feminine | Neuter |
| :--- | :--- | :--- | :--- | :--- |
| **Nom** | jarn**í** | jarn**í** | jarn**í** | jarn**í** |
| **Gen** | jarn**ího** | jarn**ího** | jarn**í** | jarn**ího** |
| **Dat** | jarn**ímu** | jarn**ímu** | jarn**í** | jarn**ímu** |
| **Acc** | jarn**ího** | jarn**í** | jarn**í** | jarn**í** |
| **Voc** | jarn**í**! | jarn**í**! | jarn**í**! | jarn**í**! |
| **Loc** | jarn**ím** | jarn**ím** | jarn**í** | jarn**ím** |
| **Instr** | jarn**ím** | jarn**ím** | jarn**í** | jarn**ím** |

### Plural Declension

| Case | All Genders |
| :--- | :--- |
| **Nom** | jarn**í** |
| **Gen** | jarn**ích** |
| **Dat** | jarn**ím** |
| **Acc** | jarn**í** |
| **Voc** | jarn**í**! |
| **Loc** | jarn**ích** |
| **Instr** | jarn**ími** |

*This makes soft adjectives much easier to learn!*
