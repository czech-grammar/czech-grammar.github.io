# Comparison (Stupňování)

Adjectives have three degrees of comparison:
1.  **Positive** (*krásný* - beautiful)
2.  **Comparative** (*krásnější* - more beautiful)
3.  **Superlative** (*nejkrásnější* - most beautiful)

## Forming the Comparative
Add the suffix **-ější**, **-ejší**, or **-ší** to the stem.

*   *chytrý* (smart) → *chytr**ější*** (smarter)
*   *zajímavý* (interesting) → *zajímav**ější***
*   *starý* (old) → *star**ší***

## Forming the Superlative
Simply add the prefix **nej-** to the Comparative form.

*   *chytrější* → ***nej**chytrější* (smartest)
*   *starší* → ***nej**starší* (oldest)

## Irregular Adjectives
Some common adjectives are irregular. Memorize them!

| Positive | Comparative | Superlative | Meaning |
| :--- | :--- | :--- | :--- |
| **dobrý** | **lepší** | **nejlepší** | good / better / best |
| **špatný** | **horší** | **nejhorší** | bad / worse / worst |
| **velký** | **větší** | **největší** | big / bigger / biggest |
| **malý** | **menší** | **nejmenší** | small / smaller / smallest |

## Comparing with "Než"
To say "than", use **než**.
*   *Petr je starší **než** Pavel.* (Peter is older than Paul.)
