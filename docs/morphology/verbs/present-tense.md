# Present Tense (Přítomný čas)

Czech verbs are divided into 5 conjugation classes based on their 3rd person singular ending.

## Class 1: -Á (Dělat)
Verbs ending in **-at** in infinitive.

| Person | Ending | Example (Dělat - to do) |
| :--- | :--- | :--- |
| Já | **-ám** | děl**ám** |
| Ty | **-áš** | děl**áš** |
| On | **-á** | děl**á** |
| My | **-áme** | děl**áme** |
| Vy | **-áte** | děl**áte** |
| Oni | **-ají** | děl**ají** |

*Others: Hledat, Čekat, Zpívat.*

## Class 2: -Í (Rozumět / Mluvit)
Verbs ending in **-ět** or **-it**.

| Person | Ending | Example (Mluvit - to speak) | Example (Rozumět - to understand) |
| :--- | :--- | :--- | :--- |
| Já | **-ím** | mluv**ím** | rozum**ím** |
| Ty | **-íš** | mluv**íš** | rozum**íš** |
| On | **-í** | mluv**í** | rozum**í** |
| My | **-íme** | mluv**íme** | rozum**íme** |
| Vy | **-íte** | mluv**íte** | rozum**íte** |
| Oni | **-í** | mluv**í** | rozum**í** |

*Others: Bydlet, Myslet, Učit se, Vařit.*

## Class 3: -JE (Kupovat / Pít)
Verbs ending in **-ovat** (change to -uj-) or short monosyllabic verbs.

| Person | Ending | Example (Kupovat - to buy) | Example (Pít - to drink) |
| :--- | :--- | :--- | :--- |
| Já | **-ji / -u** | kupu**ji** / kupu**ju** | pi**ji** / pi**ju** |
| Ty | **-ješ** | kupu**ješ** | pi**ješ** |
| On | **-je** | kupu**je** | pi**je** |
| My | **-jeme** | kupu**jeme** | pi**jeme** |
| Vy | **-jete** | kupu**jete** | pi**jete** |
| Oni | **-jí / -jou** | kupu**jí** / kupu**jou** | pi**jí** / pi**jou** |

*Note: Forms like "kupuju" are colloquial but very common. "Kupuji" is formal.*
*Others: Studovat, Pracovat, Žít, Psát (Píše).*

## Class 4: -E (Nést / Brát)
Hard stem verbs.

| Person | Ending | Example (Nést - to carry) | Example (Brát - to take) |
| :--- | :--- | :--- | :--- |
| Já | **-u** | nes**u** | ber**u** |
| Ty | **-eš** | nes**eš** | ber**eš** |
| On | **-e** | nes**e** | ber**e** |
| My | **-eme** | nes**eme** | ber**eme** |
| Vy | **-ete** | nes**ete** | ber**ete** |
| Oni | **-ou** | nes**ou** | ber**ou** |

*Others: Číst (Čtu), Jít (Jdu), Jet (Jedu).*

## Class 5: -NE (Tisknout)
Verbs ending in **-nout**.

| Person | Ending | Example (Tisknout - to print) |
| :--- | :--- | :--- |
| Já | **-nu** | tisk**nu** |
| Ty | **-neš** | tisk**neš** |
| On | **-ne** | tisk**ne** |
| My | **-neme** | tisk**neme** |
| Vy | **-nete** | tisk**nete** |
| Oni | **-nou** | tisk**nou** |

*Others: Zapomenout, Zvednout.*

## Irregular Verb: Být (To be)
| Person | Form |
| :--- | :--- |
| Já | **jsem** |
| Ty | **jsi** |
| On | **je** |
| My | **jsme** |
| Vy | **jste** |
| Oni | **jsou** |
