# Past Tense (Minulý čas)

The past tense is formed using the **l-participle** of the main verb + the auxiliary verb **být** (to be).

## Formation

1.  Take the infinitive (*dělat*).
2.  Remove the **-t** (*děla-*).
3.  Add the **-l** ending based on gender.

| Gender | Ending | Example (dělat) |
| :--- | :--- | :--- |
| Masculine | **-l** | *dělal* |
| Feminine | **-la** | *dělala* |
| Neuter | **-lo** | *dělalo* |
| Plural (Ma) | **-li** | *dělali* |
| Plural (F/Mi) | **-ly** | *dělaly* |
| Plural (N) | **-la** | *dělala* |

## Conjugation with Auxiliary

The auxiliary verb *být* is used in the 1st and 2nd person. **It is omitted in the 3rd person.**

| Person | Auxiliary | Main Verb (Masc.) | Full Sentence |
| :--- | :--- | :--- | :--- |
| Já | **jsem** | dělal | *Dělal jsem to.* (I did it.) |
| Ty | **jsi** | dělal | *Dělal jsi to.* (You did it.) |
| On | **-** | dělal | *Dělal to.* (He did it.) |
| My | **jsme** | dělali | *Dělali jsme to.* (We did it.) |
| Vy | **jste** | dělali | *Dělali jste to.* (You did it.) |
| Oni | **-** | dělali | *Dělali to.* (They did it.) |

## Reflexive Verbs
If the verb is reflexive (*se* or *si*), the auxiliary *jsem/jsi* often merges with it in spoken/informal Czech, but in standard writing, they stay separate or follow specific word order rules.
*   *Myl jsem se.* (I washed myself.)
