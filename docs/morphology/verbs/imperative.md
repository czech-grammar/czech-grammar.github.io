# Imperative Mood (Rozkazovací způsob)

The imperative is used for commands, requests, or strong advice. It exists for **Ty** (You sg.), **My** (Let's), and **Vy** (You pl.).

## Formation

Start with the **3rd person plural** (they) of the present tense, and remove the ending (*-ou* or *-í*).

### Group 1: Vowel Ending
If the stem ends in a single consonant.
*   *Dělat* → *dělají* → *děl-* → **Dělej!** (Do!)
    *   *Ty*: **Dělej!**
    *   *My*: **Dělejme!**
    *   *Vy*: **Dělejte!**

### Group 2: Consonant Ending
If the stem ends in two consonants.
*   *Myslet* → *myslí* → *mysl-* → **Mysli!** (Think!)
    *   *Ty*: **Mysli!**
    *   *My*: **Mysleme!**
    *   *Vy*: **Myslete!**

### Irregular Imperatives
Some common verbs are irregular:
*   **Být** (to be): *Buď! Buďme! Buďte!*
*   **Jíst** (to eat): *Jez! Jezme! Jezte!*
*   **Mít** (to have): *Měj! Mějme! Mějte!* (Have a nice day = *Měj se hezky!*)
*   **Jít** (to go): *Jdi! Jděme! Jděte!* (Or colloquially: *Běž!*)
