# Adverbs (Příslovce)

Adverbs modify verbs, adjectives, or other adverbs. Unlike adjectives, they **don't decline** — they stay the same in every context!

## Formation from Adjectives

Most adverbs are formed from adjectives:

### Hard Adjectives: -ý → -ě / -e
| Adjective | Adverb | Meaning |
| :--- | :--- | :--- |
| rychlý | **rychle** | quickly |
| pomalý | **pomalu** | slowly |
| dobrý | **dobře** | well |
| špatný | **špatně** | badly |

### After k, h, ch: -ý → -y
| Adjective | Adverb | Meaning |
| :--- | :--- | :--- |
| český | **česky** | in Czech |
| anglický | **anglicky** | in English |
| tichý | **tiše** | quietly |

### Soft Adjectives: -í → -ě
| Adjective | Adverb | Meaning |
| :--- | :--- | :--- |
| jarní | **jarně** | in a spring-like way |

## Comparison

### Comparative: add -eji / -ěji
- rychle → **rychleji** (more quickly)
- pomalu → **pomaleji** (more slowly)

### Superlative: nej- + comparative
- rychle → **nejrychleji** (most quickly)

### Irregular
| Positive | Comparative | Superlative |
| :--- | :--- | :--- |
| dobře (well) | **lépe** | **nejlépe** |
| špatně (badly) | **hůře** | **nejhůře** |
| hodně (much) | **více** | **nejvíce** |
| málo (little) | **méně** | **nejméně** |

## Common Adverbs by Type

### Time
teď (now), dnes (today), zítra (tomorrow), včera (yesterday), vždy (always), nikdy (never), často (often), někdy (sometimes)

### Place
tady/zde (here), tam (there), všude (everywhere), nikde (nowhere), nahoře (above), dole (below)

### Manner
jak (how), tak (so), dobře (well), špatně (badly), rychle (quickly), pomalu (slowly)

### Degree
velmi/moc (very), trochu (a little), dost (enough), příliš (too much), úplně (completely)
