# Vocative Case (Vokativ)

**Question**: *Oslovujeme, voláme!* (We address, we call!)

The Vocative is used exclusively for **addressing people** (or things, poetically). It is NOT optional in Czech; using the Nominative to address someone sounds rude or uneducated.

## Declension (Singular)

| Gender | Model | Ending | Example (Nom -> Voc) |
| :--- | :--- | :--- | :--- |
| **Masc An** | Pán | **-e** | *Pán -> Pan**e*** (*Petr -> Petr**e***) |
| | (Soft ending) | **-i** | *Tomáš -> Tomáš**i*** |
| | (Ending in -k, -h, -ch) | **-u** | *Marek -> Mark**u*** |
| | Muž | **-i** | *Muž -> Muž**i*** |
| | Předseda | **-o** | *Předseda -> Předsed**o*** (*Kolega -> Koleg**o***) |
| | Soudce | **-e** | *Soudce -> Soudc**e*** |
| **Fem** | Žena | **-o** | *Žena -> Žen**o*** (*Jana -> Jan**o***) |
| | Růže | **-e** | *Růže -> Růž**e*** (*Marie -> Mari**e***) |
| | Píseň | **-i** | *Píseň -> Písn**i*** |
| | Kost | **-i** | *Kost -> Kost**i*** |
| **Neut** | All | **= Nom** | *Město -> Město!* |

## Common Irregularities
*   **Syn** (Son) -> **Synu!**
*   **Bůh** (God) -> **Bože!**
*   **Člověk** (Man/Human) -> **Člověče!**

## Usage
*   *Ahoj **Petře**!* (Hi Peter!)
*   *Dobrý den, pane **Nováku**.* (Good day, Mr. Novák.)
*   *Promiňte, **slečno**.* (Excuse me, Miss.)
