# Number (Číslo)

Czech distinguishes between **Singular** (*jednotné číslo*) and **Plural** (*množné číslo*).

## Forming the Plural (Nominative)

The plural ending depends on the gender.

### Masculine Animate
*   Hard consonants: **-i** (often causes mutation of the preceding consonant).
    *   *student* → *studenti*
    *   *kluk* → *kluci* (k → c)
    *   *doktor* → *doktoři* (r → ř)
*   Soft consonants: **-i** or **-é**.
    *   *muž* → *muži*

### Masculine Inanimate
*   Hard consonants: **-y**.
    *   *hrad* → *hrady*
    *   *banán* → *banány*
*   Soft consonants: **-e**.
    *   *stroj* → *stroje*

### Feminine
*   Ending in -a: **-y**.
    *   *žena* → *ženy*
    *   *kniha* → *knihy*
*   Ending in -e: **-e** (no change) or **-ice**.
    *   *růže* → *růže*
*   Ending in consonant: **-i** or **-i**.
    *   *kost* → *kosti*

### Neuter
*   Ending in -o: **-a**.
    *   *město* → *města*
    *   *auto* → *auta*
*   Ending in -e: **-ata** (for animals) or **-e**.
    *   *kuře* → *kuřata*
    *   *moře* → *moře*
*   Ending in -í: **-í** (no change).
    *   *náměstí* → *náměstí*

## Pluralia Tantum
Some words exist only in plural, even if they refer to a single object.
*   *kalhoty* (trousers)
*   *narozeniny* (birthday)
*   *dveře* (door)
