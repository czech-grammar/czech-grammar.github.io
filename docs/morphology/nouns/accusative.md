# Accusative Case (Akuzativ)

**Question**: *Koho? Co?* (Whom? What?)

The Accusative is the most common case after Nominative. It marks the **Direct Object**.

## Declension (Singular)

| Gender | Model | Ending | Example (Nom -> Acc) |
| :--- | :--- | :--- | :--- |
| **Masc An** | Pán | **-a** | *Pán -> Pán**a*** (= Genitive) |
| | Muž | **-e** | *Muž -> Muž**e*** (= Genitive) |
| | Předseda | **-u** | *Předseda -> Předsed**u*** |
| | Soudce | **-e** | *Soudce -> Soudc**e*** |
| **Masc In** | All | **= Nom** | *Hrad -> Hrad*, *Stroj -> Stroj* |
| **Fem** | Žena | **-u** | *Žena -> Žen**u*** |
| | Růže | **-i** | *Růže -> Růž**i*** |
| | Píseň | **= Nom** | *Píseň -> Píseň* |
| | Kost | **= Nom** | *Kost -> Kost* |
| **Neut** | All | **= Nom** | *Město -> Město*, *Moře -> Moře* |

*Key Rule: For Masculine Inanimate and Neuter, Accusative = Nominative. For Feminine "Žena", -a becomes -u.*

## Usage

### 1. Direct Object
*   *Mám **nové auto**.* (I have a new car.)
*   *Vidím **psa**.* (I see a dog.)
*   *Čtu **knihu**.* (I am reading a book.)

### 2. Motion Towards (Kam?)
Used with prepositions **na, pod, nad, před, za** when implying **motion**.
*   *Jdu na **poštu**.* (I am going to the post office.)
*   *Dám to pod **stůl**.* (I will put it under the table.)

### 3. Prepositions (Always Accusative)
*   **Pro** (for): *Dárek pro **maminku**.*
*   **Přes** (across/over): *Přes **most**.*
*   **Skrz** (through): *Skrz **les**.*
*   **Za** (in time): *Za **hodinu**.*

### 4. Verbs (Valency)
Most transitive verbs take the Accusative.
*   *Hledat* (to look for)
*   *Potřebovat* (to need)
*   *Znat* (to know someone)
*   *Milovat* (to love)
