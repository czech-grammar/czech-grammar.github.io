# Genitive Case (Genitiv)

**Question**: *Koho? Čeho?* (Of whom? Of what?)

The Genitive is the "Of" case. It indicates possession, quantity, and follows many prepositions.

## Declension (Singular)

| Gender | Model | Ending | Example (Nom -> Gen) |
| :--- | :--- | :--- | :--- |

| | Muž | **-e** | *Muž -> Muž**e*** |

| | Soudce | **-e** | *Soudce -> Soudc**e*** |

## Usage

### 1. Possession (Of)
*   *Auto **mého otce**.* (My father's car.)
*   *Barva **auta**.* (The color of the car.)

### 2. Quantity (Partitive)
Used after numbers (5+) and quantity words.
*   *Pět **piv**.* (Five beers - Gen. Plural)
*   *Hodně **lidí**.* (A lot of people.)
*   *Trochu **vody**.* (A little bit of water.)
*   *Láhev **vína**.* (A bottle of wine.)

### 3. Prepositions
*   **Od** (from): *Od **Petra**.*
*   **Do** (into/to): *Do **Prahy**.*
*   **Z/Ze** (out of/from): *Z **domu**.*
*   **Bez** (without): *Bez **cukru**.*
*   **U** (at/by): *U **babičky**.*
*   **Během** (during): *Během **dne**.*
*   **Kromě** (except): *Kromě **tebe**.*
*   **Vedle** (next to): *Vedle **školy**.*
*   **Okolo/Kolem** (around): *Kolem **stolu**.*

### 4. Verbs (Valency)
*   **Ptát se** (to ask): *Ptám se **učitele**.*
*   **Bát se** (to be afraid of): *Bojím se **psa**.*
*   **Všímat si** (to notice): *Všiml si **chyby**.*
