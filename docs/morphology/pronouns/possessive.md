# Possessive Pronouns (Přivlastňovací zájmena)

These pronouns answer the question **Čí?** (Whose?).

## The Forms

| Person | Pronoun | Meaning |
| :--- | :--- | :--- |
| Já | **můj** | my |
| Ty | **tvůj** | your |
| On | **jeho** | his (Indeclinable!) |
| Ona | **její** | her |
| To | **jeho** | its (Indeclinable!) |
| My | **náš** | our |
| Vy | **váš** | your |
| Oni | **jejich** | their (Indeclinable!) |

## Declension of Můj (My)
*Tvůj (Your) and Svůj (Own) follow the exact same pattern.*

### Singular

| Case | Masc An | Masc In | Fem | Neut |
| :--- | :--- | :--- | :--- | :--- |
| **Nom** | můj | můj | moje / má | moje / mé |
| **Gen** | mého | mého | mojí / mé | mého |
| **Dat** | mému | mému | mojí / mé | mému |
| **Acc** | mého | můj | moji / mou | moje / mé |
| **Loc** | mém | mém | mojí / mé | mém |
| **Instr** | mým | mým | mojí / mou | mým |

### Plural

| Case | Masc An | Others |
| :--- | :--- | :--- |
| **Nom** | moji / mí | moje / mé (N: moja / má) |
| **Gen** | mých | mých |
| **Dat** | mým | mým |
| **Acc** | moje / mé | moje / mé (N: moja / má) |
| **Loc** | mých | mých |
| **Instr** | mými | mými |

## Declension of Náš (Our)
*Váš (Your) follows the exact same pattern.*
