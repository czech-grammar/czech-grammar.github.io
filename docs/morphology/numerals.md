# Numerals (Číslovky)

Czech numerals are more complex than English because numbers 1-4 decline (change form based on case) and agree with nouns in gender!

## Cardinal Numbers (Základní číslovky)

### 1-10
| Number | Czech |
| :--- | :--- |
| 1 | jeden/jedna/jedno |
| 2 | dva/dvě |
| 3 | tři |
| 4 | čtyři |
| 5 | pět |
| 6 | šest |
| 7 | sedm |
| 8 | osm |
| 9 | devět |
| 10 | deset |

### 11-20
jedenáct, dvanáct, třináct, čtrnáct, patnáct, šestnáct, sedmnáct, osmnáct, devatenáct, dvacet

### Tens
20 dvacet, 30 třicet, 40 čtyřicet, 50 padesát, 60 šedesát, 70 sedmdesát, 80 osmdesát, 90 devadesát, 100 sto

## The 5+ Rule

Numbers **5 and higher** take the **genitive plural** of the noun!

| Number | Example | Translation |
| :--- | :--- | :--- |
| 1 muž | jeden **muž** | one man (nominative) |
| 2 muži | dva **muži** | two men (nominative plural) |
| 3-4 muži | tři **muži** | three men (nominative plural) |
| 5+ mužů | pět **mužů** | five men (genitive plural!) |

## Declension of 1 (Jeden)

Declines like the demonstrative *ten/ta/to*:

| Case | Masc | Fem | Neut |
| :--- | :--- | :--- | :--- |
| Nom | jeden | jedna | jedno |
| Gen | jednoho | jedné | jednoho |
| Dat | jednomu | jedné | jednomu |
| Acc | jednoho/jeden | jednu | jedno |
| Loc | jednom | jedné | jednom |
| Instr | jedním | jednou | jedním |

## Declension of 2 (Dva/Dvě)

| Case | Masc/Neut | Fem |
| :--- | :--- | :--- |
| Nom/Acc | dva | dvě |
| Gen/Loc | dvou | dvou |
| Dat/Instr | dvěma | dvěma |

*Note: oba/obě (both) declines the same way.*

## Ordinal Numbers (Řadové číslovky)

Ordinals decline like adjectives:
- 1st **první** (soft)
- 2nd **druhý** (hard)
- 3rd **třetí** (soft)
- 4th **čtvrtý** (hard)
- 5th **pátý**, 6th **šestý**, 7th **sedmý**, 8th **osmý**, 9th **devátý**, 10th **desátý**

*Třetí vidím dům.* (I see the third house.)
