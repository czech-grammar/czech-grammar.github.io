# Human Body (Lidské tělo)

## Main Parts

| Czech | English |
| :--- | :--- |
| **Hlava** | Head |
| **Krk** | Neck |
| **Rameno** | Shoulder |
| **Ruka** | Arm / Hand |
| **Prst** | Finger |
| **Hrudník** | Chest |
| **Břicho** | Belly / Stomach |
| **Záda** | Back |
| **Noha** | Leg / Foot |
| **Koleno** | Knee |

## Face (Obličej)
*   **Oko** (Eye) - pl. *Oči*
*   **Ucho** (Ear) - pl. *Uši*
*   **Nos** (Nose)
*   **Ústa / Pusa** (Mouth)
*   **Zub** (Tooth) - pl. *Zuby*
*   **Vlasy** (Hair)

## At the Doctor (U lékaře)
*   **Bolí mě hlava.** (My head hurts.)
*   **Mám horečku.** (I have a fever.)
*   **Jsem nemocný.** (I am sick.)
*   **Potřebuji lék.** (I need medicine.)
