# Food and Drink (Jídlo a pití)

## Basics

| Czech | English |
| :--- | :--- |
| **Chléb / Chleba** | Bread |
| **Rohlík** | Bread roll (Czech staple) |
| **Máslo** | Butter |
| **Sýr** | Cheese |
| **Šunka** | Ham |
| **Maso** | Meat |
| **Kuře** | Chicken |
| **Vepřové** | Pork |
| **Hovězí** | Beef |
| **Ryba** | Fish |
| **Vejce** | Egg |

## Drinks (Nápoje)
*   **Voda** (Water) - *Perlivá* (Sparkling) / *Neperlivá* (Still)
*   **Pivo** (Beer) - *Světlé* (Light) / *Černé* (Dark)
*   **Víno** (Wine) - *Červené* (Red) / *Bílé* (White)
*   **Káva** (Coffee)
*   **Čaj** (Tea)
*   **Mléko** (Milk)

## In a Restaurant (V restauraci)
*   **Jídelní lístek** (Menu)
*   **Dám si...** (I will have...)
*   **Účet, prosím.** (The bill, please.)
*   **Dobrou chuť!** (Bon appétit!)
*   **Na zdraví!** (Cheers!)
