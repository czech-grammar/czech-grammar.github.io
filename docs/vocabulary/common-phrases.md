# Common Phrases (Běžné fráze)

## Greetings

| Czech | English | Context |
| :--- | :--- | :--- |
| **Dobrý den** | Good day / Hello | Formal, standard greeting. |
| **Ahoj / Čau** | Hi / Bye | Informal, friends only. |
| **Dobré ráno** | Good morning | Before 9-10 AM. |
| **Dobrý večer** | Good evening | After 6 PM. |
| **Na shledanou** | Goodbye | Formal. |

## Basic Conversation

*   **Jak se máte?** (How are you? - Formal)
*   **Jak se máš?** (How are you? - Informal)
*   **Mám se dobře.** (I am fine.)
*   **Jmenuji se...** (My name is...)
*   **Těší mě.** (Nice to meet you.)
*   **Nerozumím.** (I don't understand.)
*   **Mluvíte anglicky?** (Do you speak English?)
