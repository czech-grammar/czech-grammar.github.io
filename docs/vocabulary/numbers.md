# Numbers (Čísla)

## 0 - 10

| # | Czech | Pronunciation |
| :--- | :--- | :--- |
| 0 | **nula** | |
| 1 | **jeden** (M), **jedna** (F), **jedno** (N) | |
| 2 | **dva** (M), **dvě** (F/N) | |
| 3 | **tři** | |
| 4 | **čtyři** | |
| 5 | **pět** | |
| 6 | **šest** | |
| 7 | **sedm** | *sedum* |
| 8 | **osm** | *osum* |
| 9 | **devět** | |
| 10 | **deset** | |

## 11 - 19
Add **-náct**.
*   11: **jedenáct**
*   12: **dvanáct**
*   13: **třináct**
*   ...
*   19: **devatenáct**

## 20 - 100
Multiples of ten end in **-cet** or **-desát**.
*   20: **dvacet**
*   30: **třicet**
*   40: **čtyřicet**
*   50: **padesát**
*   60: **šedesát**
*   ...
*   100: **sto**

## Ordinals (1st, 2nd...)
Behave like adjectives (hard).
*   1st: **první** (Soft!)
*   2nd: **druhý**
*   3rd: **třetí** (Soft!)
*   4th: **čtvrtý**
*   5th: **pátý**
