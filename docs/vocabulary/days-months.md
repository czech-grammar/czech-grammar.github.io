# Days and Months

## Days of the Week (Dny v týdnu)

| Day | Czech | Origin/Meaning |
| :--- | :--- | :--- |
| Monday | **Pondělí** | After Sunday (Po neděli) |
| Tuesday | **Úterý** | |
| Wednesday | **Středa** | Middle (Střed) |
| Thursday | **Čtvrtek** | Fourth (Čtvrtý) |
| Friday | **Pátek** | Fifth (Pátý) |
| Saturday | **Sobota** | Sabbath |
| Sunday | **Neděle** | No work (Nedělat) |

*Note: Days are capitalized only at the start of a sentence, unlike in English! (Except in this table for clarity).*

## Months (Měsíce)
Czech months have unique Slavic names related to nature, not Latin ones like in English.

1.  **Leden** (Ice)
2.  **Únor** (Submerge/Ice breaking)
3.  **Březen** (Birch tree)
4.  **Duben** (Oak tree)
5.  **Květen** (Blossom)
6.  **Červen** (Red/Worms?)
7.  **Červenec** (Little Red)
8.  **Srpen** (Sickle/Harvest)
9.  **Září** (Shining/Rutting season)
10. **Říjen** (Rutting season)
11. **Listopad** (Leaf fall)
12. **Prosinec** (Pig slaughter/Asking?)

## Seasons (Roční období)
*   **Jaro** (Spring)
*   **Léto** (Summer)
*   **Podzim** (Autumn)
*   **Zima** (Winter)
